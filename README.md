# VRC / HRV RRi Analyzer Pro v4.1

Versión con:
- selección con ratón sobre el gráfico RRi,
- ventanas libres Basal + E1-E6 + R1-R3,
- comparación entre registros por fases seleccionadas,
- columnas verticales agrupadas por fase y registro,
- líneas de tendencia superpuestas por registro,
- comparación multiparámetro combinada o una gráfica por parámetro,
- RRi superpuesto por fase,
- exportación Excel/ZIP.

Ejecutar:
```bash
streamlit run app.py
```
